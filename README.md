# bz-ios-button

React Native IOS style button

## Installation

```sh
npm install bz-ios-button
```

## Usage

```js
import IosButton from 'bz-ios-button';

// ...

<IosButton
  title="Hello World"
  size="md"
  bgColor="#007AFF"
  type="filled"
  textWeight="normal"
  textColor="#FFFFFF"
  onPress={() => console.log('Pressed')}
/>;
```
